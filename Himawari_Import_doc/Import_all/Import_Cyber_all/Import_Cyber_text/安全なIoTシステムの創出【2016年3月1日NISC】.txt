 新たな「サイバーセキュリティ戦略」について（全体構成）
１ サイバー空間
に係る認識

サイバー空間は、「無限の価値を産むフロンティア」である人工空間であり、人々の経済社会の活動基盤
あらゆるモノがネットワークに連接され、実空間とサイバー空間との融合が高度に深化した「連接融合情報社会(連融情報社会)」が到来
同時に、サイバー攻撃の被害規模や社会的影響が年々拡大、脅威の更なる深刻化が予想

２ 目的

「自由、公正かつ安全なサイバー空間」を創出・発展させ、もって 「経済社会の活力の向上及び持続的発展」、
「国民が安全で安心して暮らせる社会の実現」 、「国際社会の平和・安定及び我が国の安全保障」 に寄与する。
